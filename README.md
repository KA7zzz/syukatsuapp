# 就活管理アプリ

就活に関する情報をNotionで管理していたが、操作性に課題を感じたため、自分で使いやすいWebアプリを開発した。

## 機能

- **企業管理** : 応募企業の基本情報（業界・URL・選考ステージ・結果）を一元管理
- **面接スケジュール管理** : 面接日時・場所・担当者・URLをメモとともに記録
- **タスク管理** : 企業ごとにタスクを登録し、完了/未完了のステータス管理
- **書類管理** : 提出書類の状況・期限をトラッキング
- **メモ管理** : 企業研究や選考メモを自由に記録
- **カレンダー表示** : 応募日・面接日程をカレンダーで一覧表示
- **ユーザー認証** : 登録・ログイン機能によるデータの個人管理

## 技術スタック

| 領域 | 技術 |
|------|------|
| バックエンド | Python / Flask / Flask-SQLAlchemy / Flask-Login |
| データベース | PostgreSQL (Neon) |
| デプロイ | Render / Gunicorn |
| 開発環境 | Cursor |

## 開発の背景

就活情報をNotionで管理していたが、テーブルの入力や日時設定の操作性に課題を感じた。また、大学院での研究・PBL活動でシステム開発に取り組んでいたこともあり、自身で初めてのWebアプリ開発に挑戦した。

設計・実装・デプロイまで一気通貫で行い、友人数名に実際に利用してもらえるレベルまで完成させた。動作速度等の改善余地は残るが、継続的に改善中。
